# cc11ty

**Eleventy + Claude Code** で素早くサイトを構築するためのテンプレートリポジトリ。

Claude Code のスキルと組み合わせることで、デザイン生成からページ作成まで自然言語で操作できます。

次のようなサイトを30分程度で作成できます（主に、待ち時間）。

- [こだわりの自然由来の石鹸のサイト例](https://sora-soap-demo.pages.dev/)
- [科学的アプローチによる英語学習コーチングのサイト例](https://c11ty-demo-englishcoarch.pages.dev/)

## 必要環境

- Node.js 22 以上
- [Claude Code](https://claude.ai/code)

## セットアップ

```bash
git clone https://github.com/toiee-kameda/cc11ty.git
cd cc11ty
npm install
npm run start
```

開発サーバーが起動したら http://localhost:8080 を開きます。

## 開発コマンド

| コマンド | 内容 |
|---|---|
| `npm run start` | 開発サーバー起動（ライブリロード） |
| `npm run build` | 本番ビルド（`_site/` に出力） |
| `npm run debug` | デバッグモードでビルド |

## プロジェクト構成

```
src/
  _layouts/
    base.njk        # ベースレイアウト（全ページ共通）
    post.njk        # ブログ記事レイアウト
  _includes/        # 再利用パーツ
  _data/            # グローバルデータ
  index.njk         # トップページ
  posts/            # ブログ記事（Markdown）
project-docs/       # デザインプロトタイプ（HTML）置き場
.claude/
  skills/           # Claude Code カスタムスキル
_site/              # ビルド出力（git 管理外）
```

- テンプレートエンジン: Nunjucks (`.njk`) と Markdown
- Input: `src/` → Output: `_site/`

## テーマ（デザイン）作成

Claude Code を使って、以下のように進めます。

### 1. デザインモックアップを作る

まず、テーマとして採用するデザインモックアップとしてのHTMLファイルを作成します。

```
Webサイトのデザインを決めるために、  @project-docs/index.html にワンページ完結のWebページを作ってください。内容は、以下の通りです。

（ここに作りたいサイトの目的、情報などを箇条書きで記載）
```

`project-docs/index.html` に HTML プロトタイプが保存されます。

### 2. テーマ（レイアウト）に変換する

プロトタイプを Eleventy のベースレイアウトに変換します。

```
/set-theme
```

project-docs/index.htmlを選ぶことで、`src/_layouts/base.njk` に変換されます。

時々、ナビゲーションなどが意図通りに再現されません。その場合は、以下のように調整を依頼します。

```
ナビやフッターのリンクをクリックしたとき、スムーズスクロールになりません。修正してください。
ナビと、ブログ記事のヒーローセクションが重なっていません。重なるように修正してください。
```

### 3. ブログ記事のデザインを決める。

次のような指示をします。好みで変更してください。

```
ブログ記事のためのレイアウトを作ってください。

- @src/_layouts/base.njk を参考にしてください 
- このサイトのデザインは、ヒーローセクションにナビが重なるものなので、ブログ記事の最初はヒーローセクションにする
- ヒーローセクションの背景画像は、デフォルト値（ unsplash-image-finder で探して、セットしてください）で設定。ユーザー側で変更できるようにする
- ヒーローセクションは、高さが大きくなりすぎないようにする。ブログのタイトルが読みやすくする
- ブログ記事の部分は、読みやすい記事にする
- 記事の見た目をチェックできるように、Markdown 書式を複数書き込んだ hello world のブログ記事をセットする

実装するために情報が足りなければ、尋ねてください。
```

ブログ記事のデザインが意図通りにならないときは、微調整を依頼します。

```
ナビと、ブログ記事のヒーローセクションが重なっていません。重なるように修正してください。また、ヒーローセクションの画像が無効です。 /unsplash-image-finder で検索して、当てはめてください。同様に、記事内の画像も無効です。 /unsplash-image-finder で検索してください。 
```

```
デフォルトの画像を変更したいです。 /unsplash-image-finder で検索して、候補を提示してください。その中から選びます
```


### 4. ブログ記事一覧のデザインを作成

次のような指示をします。

```
ブログ記事一覧を作りたいので、 あと ５つ、このサービスらしい記事を創造して作成してください。その後、ブログ記事一覧ページを作成してください。なお、hero画像を unsplash-image-finder  で探して設定してください。
```

微調整は、次のように行います。

```
ブログ一覧のナビと、ブログ記事のヒーローセクションが重なっていません。重なるように修正してください。
```

さらに、ナビに「ブログ一覧」へのリンクを挿入させます。

```
ブログ一覧へのリンクをナビ部分と、フッターに追加してください
```

### おまけ

基本は、ワンページで、ブログ機能だけを追加する運用をする場合、どのブログ記事からも、ワンページ（トップページ）のセクションへリンクされるように、以下のように依頼します。

```
ナビ、フッターのリンクのうち、 `#hogehoge` となっているものを、 `/#hogehoge` としてください。どのページでも、トップページの特定のセクションに移動できるようにしたいです。
```


## Claude Code スキル

| スキル | 説明 | 起動ワード例 |
|---|---|---|
| `one-page-site-builder` | ワンページHTMLサイトを生成して `project-docs/` に保存 | 「LP作って」「ポートフォリオ」 |
| `set-theme` | `project-docs/` のHTMLを `base.njk` に変換 | 「テーマ設定して」「デザイン適用」 |
| `create-page` | 固定ページ（`.njk`）を作成 | 「Aboutページ」「お問い合わせ」 |
| `unsplash-image-finder` | Unsplash から画像を検索・最適化 | 自動呼び出し |

## ファイル保存規則

| 種類 | 保存先 |
|---|---|
| デザインプロトタイプ | `project-docs/*.html` |
| ベースレイアウト | `src/_layouts/base.njk` |
| 固定ページ | `src/[ページ名].njk` |
| ブログ記事 | `src/posts/[記事名].md` |
| 静的アセット | `src/assets/` |
